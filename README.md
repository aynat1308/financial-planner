# Financial Independence Planner

A comprehensive tool for planning your path to financial independence.

## Features
- Income & Expense Tracking
- Asset & Debt Management
- Financial Independence Calculation
- Multi-year Projections
- PDF & Excel Export
- Auto-save to LocalStorage

## Quick Deploy to Vercel

### Option 1: Deploy via Vercel Dashboard (Easiest)

1. **Create a Vercel Account**
   - Go to https://vercel.com
   - Click "Sign Up"
   - Sign up with GitHub (recommended) or email

2. **Upload Your Project**
   - Download this entire folder
   - Go to https://vercel.com/new
   - Click "Browse" and select this folder
   - Click "Deploy"
   - Done! Your app is live in ~30 seconds

### Option 2: Deploy via GitHub (Recommended for Updates)

1. **Create GitHub Account** (if you don't have one)
   - Go to https://github.com/signup

2. **Create New Repository**
   - Go to https://github.com/new
   - Name: `financial-planner`
   - Make it Private (if you want)
   - Click "Create repository"

3. **Upload Files**
   - On the repository page, click "uploading an existing file"
   - Drag all files from this folder
   - Click "Commit changes"

4. **Deploy to Vercel**
   - Go to https://vercel.com/new
   - Click "Import Git Repository"
   - Select your `financial-planner` repo
   - Click "Deploy"
   - Done!

### Option 3: Deploy via Command Line (For Developers)

```bash
# Install Vercel CLI
npm install -g vercel

# Navigate to this folder
cd path/to/deployment-package

# Deploy
vercel

# Follow the prompts, then your app is live!
```

## Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
